## 对比SRAM与DRAM
- 动态RAM是靠MOS电路中的栅极电容来记忆信息的。由于电容上的电荷会泄漏,需要定时给与补充,所以动态 RAM需要 设置 刷新 电路。
- RAM需要刷新，而ROM不需要刷新。
- **破坏性读出**
	- **破坏性读出**即所**读**信息在**读出**时即被破坏。 如果需要保留这信息，可以再生 (重写)，再生是由电路自动进行的。
- **非破坏性读出**
	- 读出存储器的内容时**不会改变其内容**，因而就不需像磁芯存储器那样要把数据重新写到存储器中去。  读出时不需要再生信息，它允许我们多次读出所需要的同一个数据
	- 有两个基本优点：一是缩短工作周期，二是提高可靠性
|            | SRAM                   | DRAM                                                 |
| ---------- | ---------------------- | ---------------------------------------------------- |
| 存储信息   | 双稳态触发器(六晶体管) | 存储元电路中栅极电容上的电荷(单晶体管)               |
| 破坏性读出 | 否                     | 是                                                   |
| 刷新       | 不需要                 | 需要,电容上的电荷只能维持1-2ms,通常取2ms作为刷新周期 |
| 送行列地址 | 同时送                 | 分两次送                                             |
| 运行速度   | 快                     | 慢                                                   |
| 集成度     | 低                     | 高                                                   |
| 存储成本   | 高                     | 低                                                   |
| 主要用途   | 高速缓存               | 主机内存                                             |
