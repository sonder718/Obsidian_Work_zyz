# 特点
1. 综合统一
- 集数据定义语言,数据操控语言,数据控制语言于一体
- 能独立完成数据库生命周期中的全部活动
- 投入运行后可随时修改模式,不影响数据库的运行
- 数据操作符统一
2. 高度非过程化:只要提出做什么,无指定存储路径
3. 面向集合的操作方式,操作对象是元组的集合
4. 多种使用方式,可独立,可嵌入
5. 简洁
