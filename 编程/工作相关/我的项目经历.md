##### 基于树莓派的智能阳台系统
- 后端:基于nodejs的智能阳台系统
	- 功能
		- 进行数据分析与预警处理
	- 过程
		- 前端发送查询树莓派信息指令报文
		- 后端接收向OneNet平台发送请求
		- 接收到OneNet平台数据
	- 技术
		- 使用 node.js 的 Express 框架开发
- 前端:基于vue的web端管理系统
	- 网页客户端以 vue-element-admin 为基础，前端使用 VUE+element
- MQTT协议,OneNet平台
	- ONENET 平台提供开放的 API 接口，用户可以通过 HTTP/HTTPS 调用，进行设备管理，数据查 询，设备命令交互等操作，在 API 的基础上，根据自己的个性化需求搭建上层应用。
	- 感知层的数据信息通过WIFI无线网络与MQTT协议接入中国移动ONENet平台与腾讯云数据库
	- 而用户也可以在ONENET平台上下发浇水，开空调等“阳台环境控制命令”到树莓派实现对树莓派的远程控制。
- 树莓派4B
	- 采用树莓派4B开发板作为嵌入式网关服务器，NodeMCU作为底层控制板，结合各种传感器实现对阳台植株土壤湿度，阳台内外温湿度等数据的采集
	- 在树莓派终端利用**python的**连接MQTT服务器后，每间隔5秒循环发布传感器获取的“阳台环境状况信息”到ONENET平台并将数据存储在腾讯云数据库中。
